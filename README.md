## Automation Framework Overview

### Tech Stack
- Java
- Selenium
- Cucumber (BDD)
- TestNG
- Rest Assured

### Structure
- UI Automation: Page Object Model
- API Automation: Rest Assured Skeleton
- Config Driven: properties file

### How to Run
1. Update config.properties
2. Run TestRunner
3. API tests run via TestNG

### Covered Scenarios
- DP1 slider validation
- API GET user validation (skeleton)

### Notes
Some sites block automation (WAF/CAPTCHA).
Framework demonstrates scalable design rather than site-specific bypassing.
